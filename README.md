# ACSOS Artifact - Experimental Repository

This repository contains the experimental artifact associated with our paper submitted to **ACSOS**.

# Experiments for: C²FL: Clustered Continual Federated Learning under Spatial and Temporal Drift

The artifact includes the code required to reproduce the main experimental results discussed in the paper, together with scripts to launch the experiments and regenerate the plots.

## Requirements

The project uses **Poetry** for dependency management.
Before running the experiments, please make sure that Poetry is installed on your system.

If Poetry is not already installed, you can follow the official installation instructions:

```bash
https://python-poetry.org/docs/#installation
```

Then, from the root directory of the repository, install the project dependencies with:

```bash
poetry install
```

## Running the Experiments

To simplify the evaluation process, we provide a script called `launch_all.py` in the root directory of the repository.

This script launches all the experiments required to reproduce the results. By default, the experiments are executed with a single seed in order to keep the execution time reasonable for artifact evaluation.

From the root directory, run:

```bash
poetry run python3 launch_all.py
```

Although the artifact runs the experiments with a single seed by default, the observed trends are consistent with the results and plots reported in the paper.

## Regenerating the Plots

Once the experiments have completed, the plots can be regenerated by running:

```bash
poetry run python3 src/plotter/plotter.py
```

The generated plots reproduce the same overall behavior shown in the paper.
